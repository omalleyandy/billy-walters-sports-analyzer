Analyze the matchup between the specified teams.

Usage: /analyze-matchup [team1] [team2]

This command will:
1. Fetch recent performance data for both teams
2. Check weather conditions for the game
3. Analyze betting lines and odds movements
4. Identify value opportunities
5. Generate a comprehensive matchup report

Include in your analysis:
- Offensive and defensive efficiency metrics
- Home/away splits
- Weather impact (if outdoor game)
- Injury reports
- Historical head-to-head performance
- Sharp action and line movements
